# To-Do-List
A simple to-do list made in javascript.

## FEATURES
- [X] Add Todos
- [X] Delete Todos
- [X] Toggle Todos
- [X] Delete All Todos
- [X] Toggle All Todos

## USAGE
- Pressing enter adds the to-do list item.
- Clicking the :x: on a list item deletes it.
- Clicking the other buttons sorts the list accordingly.
